A simple and intuitive travel planning application built with Streamlit to help you organize your perfect trip!

## Features

- **Trip Details Management**: Set destination, dates, number of travelers, and budget
- **Interactive Itinerary Builder**: Create day-by-day schedules with activities, times, and costs
- **Budget Tracking**: Real-time budget monitoring with expense breakdown by day
- **Packing Checklist**: Pre-populated packing list organized by categories
- **Trip Notes**: Keep track of important information, contacts, and reminders

## Installation

1. Make sure you have Python 3.8 or higher installed

2. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Running the App

Run the application with:
```bash
streamlit run app.py
```

The app will open automatically in your default web browser at `http://localhost:8501`

## How to Use

1. **Enter Trip Details** (Sidebar):
   - Enter your destination
   - Select start and end dates
   - Specify number of travelers
   - Set your total budget and currency

2. **Build Your Itinerary**:
   - Go to the "Itinerary" tab
   - Add activities with day, time, cost, and notes
   - View your organized schedule by day
   - Delete activities as needed

3. **Track Your Budget**:
   - Check the "Budget Breakdown" tab
   - See real-time spending vs. remaining budget
   - View expense breakdown by day
   - Get suggested budget allocation

4. **Create Packing List**:
   - Use the "Packing List" tab
   - Check off items as you pack
   - Pre-organized by categories

5. **Keep Notes**:
   - Use the "Notes" tab for important reminders
   - Save emergency contacts and other details

## Technologies Used

- **Streamlit**: Web application framework
- **Python**: Programming language

## Tips

- Plan your budget before adding activities
- Add activities in chronological order for better organization
- Use the notes section for emergency contacts and important information
- Check off packing items as you prepare for your trip
